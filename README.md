# System Rezerwacji Zasobów

Aplikacja do zarządzania rezerwacjami zasobów (np. sal konferencyjnych) zbudowana w architekturze full-stack z backendem Symfony i frontendem Next.js.

## Instalacja z uruchomieniem lokalnym

```bash
make dev
```

Instalacja zależności i migracje bazy danych są wykonywane automatycznie.

Aplikacja będzie dostępna pod adresami:
- Frontend: http://localhost:3000
- Backend API: http://localhost:8080


## Uruchomienie

```bash
make run
```


## Zatrzymanie

```bash
make stop
```
